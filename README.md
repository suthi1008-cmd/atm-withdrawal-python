# ATM Withdrawal System (Python)
## Description
This project is as simple ATM Withdrawal System built using Python.
It uses CSV file handling to store and update account details.
## Features
- Account login using account number and PIN
- Check account balance
- Deposit money
- Withdraw money
- Data stored using CSV file
## Technologies Used
- Python
- CSV file handling
## Files
- atm.py
- accounts.py
## How to Run
1. Install Python on your system
2. Download the project files
3. Run the program using:
   python atm.py
